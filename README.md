# Malnutrition-Dashboard-Spatial-Study

## 📊 Project Overview

This project presents interactive Excel dashboards developed to analyze Moderate Acute Malnutrition (MAM) and Severe Acute Malnutrition (SAM) across Udupi district.

The dashboards integrate healthcare data with visual analytics to identify high-burden areas and support targeted public health interventions. Additionally, the project includes village-level coverage analysis of MAM and SAM cases, enabling granular insights into regional distribution and helping in prioritizing localized interventions for improved child health outcomes.

## 🎯 Objectives

1. Monitor and track Moderate Acute Malnutrition (MAM) and Severe Acute Malnutrition (SAM) cases across regions.
2. Support the reduction of SAM cases by enabling their transition to MAM levels through data-driven insights.
3. Identify high-burden villages and Taluks for targeted interventions.
4. Analyze gender-wise distribution of malnutrition cases.
5. Enable healthcare authorities to take informed decisions for improving child nutrition outcomes.
